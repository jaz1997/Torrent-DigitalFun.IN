## ===== Installation =====
## Git

```
$ git clone https://github.com/Lunik/Lunik-Torrent.git
$ cd Lunik-Torrent
$ npm install
$ cp src/config.default src/config.json
$ nano src/config.json
```

### Run

```
$ npm start
or with forever
$ npm run forever-start
```
